# Contributing

Thank you for contributing to OptionLab.

## Development Workflow

1. Fork repository.
2. Create feature branch.
3. Write tests.
4. Commit changes.
5. Open Pull Request.

---

## Coding Standards

- Black formatting
- Type hints
- Docstrings
- Unit tests

---
